# YABP Capability Guard

A small, dependency-free JavaScript utility that detects constrained device
capabilities and shows a polite compatibility notice.

## What it does
- Checks viewport, pointer type, memory, and CPU signals
- Shows a one-time notice
- Optional redirect or block mode
- Zero configuration required

## Install
```html
<script src="https://guardjs.netlify.app/capability-guard.min.js"></script>
```
